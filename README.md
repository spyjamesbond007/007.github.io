# H型鋼擋土支撐計算及繪圖器

擋土支撐計算、開挖回填數量計算、預力換算，以及平面圖／立面圖／3D 立體示意圖繪製工具。

## 線上使用

GitHub Pages 網址：`https://<你的帳號>.github.io/<repo名稱>/`

- **手機、平板、電腦皆可使用**（版面會自動切換）
- 手機上若圖面太小，可點圖面上方的「**原尺寸(可捲動)**」按鈕，以原始解析度左右捲動細看

## 檔案說明

| 檔案 | 說明 |
|---|---|
| `index.html` | 線上版（GitHub Pages 使用此檔），約 0.27 MB，載入快 |
| `離線完整版.html` | three.js／SheetJS 已內嵌，**無需網路**，約 1.7 MB。適合工地無訊號、隨身碟、內網環境 |

## 主要功能

- 排樁／橫檔／角撐／中間水平支撐／中間柱／斜撐 數量計算
- 木襯板面積、開挖與回填土方計算
- 千斤頂預力 ⇄ 加壓錶壓 換算（對照表可自行編輯）
- 平面圖、A-A'／B-B' 立面圖、3D 立體示意圖
- 設計檢核與防呆警示
- Excel 數量表匯出、參數存檔／載入 (JSON)
